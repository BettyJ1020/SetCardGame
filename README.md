# SetCardGame Android App
Set Card Game 是一種經典的撲克牌遊戲，玩家需要在卡牌中找到符合特定條件的組合。本專案提供一個可在手機或平板上體驗 Set Card Game 的平台。
## Document
### 遊戲規則
1. 遊戲開始時，將牌堆洗牌並在「桌面」上排列出12張（或更多，如15、18張等）卡牌。
2. 點擊卡牌時，卡牌在選中/未選中狀態之間切換。
3. 當玩家選中三張符合條件的卡牌時，將它們移除，然後發出更多的卡牌，直到桌面上有12張卡牌。否則，顯示一個通知，讓玩家知道選中的卡牌不構成有效組合。
4. 如果在十二張卡牌中找不到有效組合，則再發出三張卡牌，使發出的卡牌總數達到十五張，或者更多，如有需要。
5. 這種按三張發牌並尋找組合的過程將持續，直到牌堆耗盡並且桌面上沒有更多的有效組合。
6. 點擊【Game History】，可瀏覽過去選中的組合，瀏覽目前擁有的Set
7. 點擊【Restart】，即可重新開始遊戲。
